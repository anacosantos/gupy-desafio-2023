Ferramenta de automação: [https://www.cypress.io/]

## **Instalar Cypress**
```
npm install 

```
## **Run all tests**
```
npm run cypress:open
npm run cypress:run (headless)
```

## **Run Lint & Prettier**
```
npm run lint:prettier
```
